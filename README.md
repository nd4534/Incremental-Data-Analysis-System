# Incremental Data Analysis & Insight Generation System

## Overview

This project is developed using Python in Google Colab.

The system compares historical datasets with new datasets and automatically generates:

- Statistical analysis
- Trend analysis
- Correlation analysis
- Anomaly detection
- Visual analytics
- Insight generation

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SciPy
- Google Colab

---

## Features

- Smart CSV/XLSX reading
- Missing value handling
- Duplicate removal
- Data normalization
- Statistical comparison
- Trend detection
- Correlation heatmap
- Automated insights

---

## Project Workflow

1. Upload historical dataset
2. Upload new dataset
3. Preprocess data
4. Perform statistical analysis
5. Generate visualizations
6. Detect anomalies
7. Generate final insights
